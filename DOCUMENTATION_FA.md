# سامانه هم‌اندیش
## مستندات فارسی

---

## 📋 نمای کلی سامانه

**هم‌اندیش** یک پلتفرم مشارکت شهروندی و حل مسائل جمعی است که به مردم کمک می‌کند:
- **مشکلات اجتماعی را مطرح کنند** (شهری، محیط‌زیستی، اجتماعی، ...)
- **راه‌حل‌های خلاقانه پیشنهاد دهند** برای مشکلات پیش‌آمده
- **نظر خود را بیان کنند** و نقاط قوت/ضعف را مناقشه کنند
- **رای دهند** و محبوب‌ترین‌ها را برای اولویت‌بندی شناسایی کنند

---

## 🎯 ویژگی‌های اصلی

### 1️⃣ **ورود و احراز هویت**

#### الف) ورود از طریق پیام‌رسان‌های فوری
- 🟦 **بله** (Bale Mini App)
- 💜 **ایتا** (Eitaa Mini App)
- **بدون نیاز به ثبت‌نام:** حساب خودکار از شماره تلفن ایجاد می‌شود
- **هم‌گام‌سازی:** یک شماره = یک حساب (فارسی خواه از Bale یا Eitaa وارد شوید)
- **محفوظ:** بر اساس HMAC امضاشدگی پیام‌رسان‌ها

#### ب) ورود وب (مرورگر ساده)
- شماره موبایل + کد OTP ۵ رقمی
- از طریق درگاه پیامک IPPanel
- برای دستگاه‌هایی که Bale/Eitaa ندارند

### 2️⃣ **مشکلات (Problems)**

| ویژگی | جزئیات |
|-------|--------|
| **ایجاد** | هر کاربر می‌تواند یک مشکل را شرح دهد |
| **دسته‌بندی** | زباله، ترافیک، آب، برق، معلم‌ها، ... |
| **وضعیت** | درحال بررسی / تأیید شده / رد شده |
| **رای** | 👍 مثبت / 👎 منفی (یک رای به ازای هر کاربر) |
| **نمایش** | لیست محبوب / مشکلات برگزیده (توسط مدیر) |

### 3️⃣ **راه‌حل‌ها (Solutions)**

- ✅ **یک راه‌حل به ازای هر کاربر برای هر مشکل**
- ✏️ **قابل ویرایش** در مدت ۷ روز از نوشتن
- 🕐 **نشان می‌دهد:** زمان نوشتن + زمان آخرین ویرایش
- 💬 **پاسخ‌دهی:** دیگران می‌توانند پاسخ دهند (یک پاسخ به ازای هر کاربر)
- 📌 **سنجاق کردن:** مدیران می‌توانند راه‌حل‌های مهم را سنجاق کنند

### 4️⃣ **پروفایل کاربری**

| بخش | توضیح |
|------|--------|
| **نام** | نام و نام خانوادگی (اختیاری) |
| **شماره** | مخفی شده (مثال: `0912*****89`) |
| **نمایش نام** | اختیار کاربر - نام کامل یا مخفی‌شدگی |
| **راه‌حل‌های من** | تاریخی از تمام راه‌حل‌های نوشته‌شده |

### 5️⃣ **پنل مدیریت** (مشرف سیستم)

#### تب **درحال بررسی**
- مشکلات منتظر تأیید
- ✅ تأیید یا ❌ رد کردن

#### تب **مشکلات**
- جستجو و فیلتر‌کردن
- 🔖 انتخاب **برگزیده** (نمایش در صفحه اول)
- 🗑️ حذف

#### تب **کاربران**
- 🏷️ **برچسب:** نشان‌دادن نقش کاربر (مثلاً "شهردار"، "متخصص")
- 👤 تغییر به **مدیر** / **کاربر عادی**

#### تب **آمار**
- 🔢 تعداد مشکلات، راه‌حل‌ها، رای‌ها
- 📊 فعالیت عمومی
- 💬 **تغییر دهنده:** فعال/غیرفعال کردن نظرات و پاسخ‌ها در سراسر سیستم

## 🎨 رابط کاربری

### 📱 **ظاهر موبایل‌محور**
- طراحی واکنش‌پذیر برای تمام دستگاه‌ها
- RTL (راست‌به‌چپ) کامل برای فارسی
- دایرکشن خودکار (auto-direction) برای شماره‌های انگلیسی

### 🧭 **ناوبری پایین‌ی**
- 🔔 **خانه:** صفحه اصلی (مشکلات برگزیده و محبوب)
- ✏️ **جدید:** مطرح‌کردن مشکل جدید
- 👤 **من:** پروفایل + راه‌حل‌های من
- ⚙️ **مدیریت:** (فقط برای مدیران)

### 🎨 **رنگ‌برداری**
- رنگ‌های آرام و متمایز برای راحتی چشم
- استفاده از emoji برای درک سریع‌تر

---

## 🔐 **امنیت و حریم‌خصوصی**

| بخش | جزئیات |
|------|--------|
| **احراز هویت** | Sanctum (Token-based, بدون Session) |
| **اعتبارسنجی HMAC** | تأیید امضای Bale/Eitaa بر روی سرور |
| **محافظت Session** | شناسه Messenger در localStorage — دستگاه‌های مشترک شناسایی می‌شوند |
| **مخفی‌سازی شماره** | نمایش `09XXXXXXXXXX` به جای شماره کامل |
| **بدون کوکی** | API‌های stateless |

## 🛠️ **پشتیبانی فنی**

### اپلیکیشن
- **زبان:** PHP 8.4
- **فریم‌ورک:** Laravel 11
- **معماری:** Service Layer Pattern
- **دیتابیس:** MySQL

### Frontend
- **فریم‌ورک:** Vue 3
- **بیلد:** Vite
- **State Management:** Pinia
- **Routing:** Vue Router
- **UI:** Tailwind CSS v4 (RTL)

### Deploy
- **Hosting:** Shared Hosting (menon.ir)
- **Sub-path:** `/hamandish/`
- **CI/CD:** GitHub Actions
- **API:** RESTful + JSON

## 📈 **مثال: یک مشکل از شروع تا پایان**

### سناریو: مشکل ترافیک ساعات‌پیک

```
1️⃣ کاربر علی مشکل را مطرح می‌کند
   "ترافیک شدید بین ساعت ۷-۹ صبح در بزرگراه"
   ↓
2️⃣ سیستم منتظر تأیید مدیر
   ↓
3️⃣ مدیر تأیید می‌کند
   ↓
4️⃣ مشکل در لیست عمومی ظاهر می‌شود
   ↓
5️⃣ کاربران رای می‌دهند → ۱۵۰ تا ۲۰۰ رای
   ↓
6️⃣ راه‌حل‌ها ارسال می‌شوند:
   - علیرضا: "شهرداری باید ۳ خط اتوبوس اضافی بگذارد"
   - فاطمه: "نرخ طرح ترافیک‌ها را بشناسید (زمان آفت‌پیک)"
   - رضا: "مرکز کار را از راه دور کاهش دهید"
   ↓
7️⃣ پاسخ‌ها
   - علیرضا پاسخ داد: "اتوبوس‌های ترافیک در حال حاضر پر هستند"
   - فاطمه پاسخ داد: "۲ ساعت پیک می‌مانند"
   ↓
8️⃣ مدیر می‌تواند بهترین راه‌حل را سنجاق کند
   ↓
9️⃣ بعد از حل مشکل، شهرداری وضعیت را به "حل‌شده" تغییر می‌دهد
```

## 💡 **نکات برای طراح گرافیک**

برای ایجاد **اینفوگرافی** توصیه می‌شود بر موارد زیر تاکید کنید:

### رنگ‌های پیشنهادی
- 🔵 آبی (`#2563eb`) - ورود، دکمه‌ها
- 🟦 آبی روشن‌تر (`#3b82f6`) - تعامل
- ⚪️ سفید/خاکستری - پس‌زمینه
- 🔴 قرمز (`#ef4444`) - خطا، منفی

### آیکون‌های اصلی
- 🗣️ (chat bubble) - هم‌اندیش / مشکل مطرح کردن
- 💡 (lightbulb) - راه‌حل
- 👍 (thumbs up) - رای مثبت
- 🎯 (target) - اولویت
- 👥 (people) - جامعه
- ⚙️ (gear) - مدیریت

### جریان داده
```
کاربر
  ├── مشکل مطرح می‌کند
  ├── رای می‌دهد
  ├── راه‌حل پیشنهاد می‌دهد
  ├── پاسخ می‌دهد
  └── پروفایل را تدوین می‌کند

مدیر
  ├── مشکلات را تأیید می‌کند
  ├── برگزیده انتخاب می‌کند
  ├── کاربران را برچسب‌دار می‌کند
  └── آمار را بررسی می‌کند
```

## 🎓 **راهنمای کاربر**

### برای کاربران عادی
1. وارد شوید (Bale / Eitaa / OTP)
2. مشکلات را بخوانید و رای دهید
3. راه‌حل پیشنهاد دهید یا پاسخ دهید
4. پروفایل خود را تدوین کنید

### برای مدیران
1. وارد شوید
2. به تب **مدیریت** برید
3. مشکلات را بررسی و تأیید کنید
4. برگزیده‌ها را انتخاب کنید
5. آمار و گزارش‌ها را ببینید
6. تنظیمات سامانه را تغییر دهید
