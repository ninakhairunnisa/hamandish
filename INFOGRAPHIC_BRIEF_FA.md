# هم‌اندیش - خلاصه اجرایی برای اینفوگرافی

## عنوان اصلی
**هم‌اندیش: پلتفرم مشارکت شهروندی و حل مسائل جمعی**

---

## 5 بخش اصلی

### 1️⃣ **مشکلات (Problems)**
```
هر کاربر می‌تواند:
✏️ مشکل را توصیف کند
🏷️ دسته‌بندی کند (ترافیک، آب، برق، ...)
👍 دیگران رای دهند
📊 محبوب‌ترین مشکلات معین می‌شوند
```

### 2️⃣ **راه‌حل‌ها (Solutions)**
```
هر کاربر می‌تواند:
💡 راه‌حل پیشنهاد دهد (یک راه‌حل به ازای هر مشکل)
✏️ در طول ۷ روز ویرایش کند
💬 پاسخ برای راه‌حل دریافت کند
📌 بهترین‌ها سنجاق شوند
```

### 3️⃣ **رای‌گیری (Voting)**
```
👍 مثبت - "مشکل واقعی است"
👎 منفی - "نسبت‌اً مهم نیست"
📈 اولویت‌بندی با مشارکت جمع
```

### 4️⃣ **پروفایل کاربری**
```
👤 نام + نام خانوادگی (اختیاری)
📱 شماره (مخفی: 0912*****89)
✨ نمایش نام یا مخفی‌شدگی (انتخاب کاربر)
📜 تاریخ تمام راه‌حل‌های ارسال‌شده
```

### 5️⃣ **پنل مدیریت**
```
✅ تأیید مشکلات
🔖 انتخاب برگزیده‌ها
🏷️ برچسب‌دار کردن کاربران
📊 بررسی آمار و فعالیت
💬 فعال/غیرفعال کردن نظرات
```

## جریان مشارکت (User Journey)

```
1. ورود
   ↓
2. مشاهده مشکلات + رای دادن
   ↓
3. مطرح کردن مشکل جدید (یا پیشنهاد راه‌حل)
   ↓
4. دریافت رای و پاسخ‌ها
   ↓
5. ویرایش/بهبود (تا ۷ روز)
   ↓
6. حل شدن مشکل ✅
```

## نکات کلیدی برای طراحی

|------|-------|------------|
